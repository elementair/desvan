Suscripción del sitio web, para mandar emails cuando insertamos un nuevo post. 

== Description ==

Este plugin permite que los usuarios se suscriban por email por medio de un widget o modal. Cuando nosotros insertamos un nuevo post, ese post lo manda por correo a todos los usuarios que se han suscrito, además tenemos el control de todos los emails, pudiendo eliminar, añadir, desactivar o activar. 
También podemos mandarles emails a nuestros suscriptores sin que sea tipo post.  


== Installation ==

Hay varias opciones para la instalación y configuración de este plugin.

= Carga Manualmente =

1. Descargar y descomprimir el plugin.
2. Subir carpeta 'suscriptflech' dentro del directorio '/wp-content/plugins/'.
3. Dirigase a la página de administración del plugin y activalo.

= Instalación vía área de Administración =

1. En el área de administración ir a Plugin > Add New y buscar 'suscriptflech'.
2. Click en instalar y luego en activar.

= Configura el Plugin =

1. Entra en el área de administración, Ajustes, Suscriptflech y ahí encontraras todas las pestañas para cambiar la configuración del plugin desde cambiar el logo por el de tu empresa como activar el modal y el recaptcha.
2. También al activar el plugin se crea una página automática, para cuando el usuario confirma la suscripción vía email, le lleva a esa dicha página para mostrarle un mensaje de suscripción realizada con éxito. Podrás cambiar la descripción o cuerpo de mensaje teniendo cuidado de no eliminar la clase que viene por defecto y sin cambiar el título de la página. 

= Como usar el widget =

1. Ir a Appearance > Widgets y arrastra el Widget 'Formulario de suscripción' a la barra lateral principal.
2. Ingrese el titulo que desee.

== Frequently Asked Questions == 

= Como cambio el logo del correo que envío =

Entra en el área de administración, Ajustes, Suscriptflech y luego cambia el logo por el de tú empresa para que cuando les envie un email les salga tú logo.
